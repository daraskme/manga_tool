# 漫画セリフ入れツール

画像フォルダを読み込んで縦書き・横書きのセリフをレイヤー形式で追加し、JPG に書き出すツール。

## 必要なもの

- Python 3.8 以上
- PyQt6（初回起動時に自動インストール）

## 起動方法

`manga_serif_tool.bat` をダブルクリック。  
初回のみ仮想環境の作成と PyQt6 のインストールが走る（数十秒）。

## 使い方

1. 画像フォルダをウィンドウにドラッグ＆ドロップ
2. ツールバーから「縦書き追加」または「横書き追加」でセリフ配置
3. セリフをダブルクリックで編集、ドラッグで移動
4. ページ移動・ウィンドウを閉じると自動保存

### 保存先

元フォルダと同じ場所に `<フォルダ名>_serif/` が作られ、  
JPG ファイルと編集データ `edits.json` が保存される。  
次回同じフォルダを開くと編集内容が復元される。

## 操作

| 操作 | 動作 |
|---|---|
| ホイール | ページ送り |
| Ctrl + ホイール | 拡大縮小 |
| ホイールクリック + ドラッグ | パン（スクロール） |
| ダブルクリック | テキスト編集 |
| 右クリック | 複製 / 削除 |
| Delete | 選択中のセリフを削除 |
| ← / → | ページ移動 |
| Ctrl + 0 | 画面に合わせる |

### 縦書き入力ダイアログ

- 通常の Enter はそのまま入力
- **Ctrl + Enter** で列の改行（新しい列を左側に追加）

## ツールバー

| 項目 | 説明 |
|---|---|
| 横書き追加 / 縦書き追加 | セリフを新規追加 |
| レイヤーパネル | 右パネルの表示/非表示 |
| フォント / サイズ | 選択中のセリフに即時反映 |
| 太字 / 色 | 選択中のセリフに即時反映 |
| 文字間 | 文字と文字の間隔（px） |
| 行間 | 列（縦書き）または行（横書き）の間隔（px） |
| 縦↔横切替 | 選択中のセリフの方向を反転 |

## レイヤーパネル

- チェックボックスでセリフの表示/非表示
- クリックでシーン上のセリフを選択
- 右クリックで複製 / 削除

## ファイル構成

```
manga_tool/
├── manga_serif_tool.bat   # 起動スクリプト（仮想環境の管理も兼ねる）
├── manga_serif_tool.py    # 本体
└── .venv_manga_serif/     # 仮想環境（自動生成、git 管理外）
```
